HTML & CSS Summary

HTML Summary

HTML(HyperText Markup Language) is used to define our page content, structure and meaning. You don't use it for styling purposes. Use CSS instead!

  • HTML uses elements to describe (annotate) content

    HTML element typically have an opening tag, content and a closing tag

  • You can have void (empty) elements like images
  • You can also configure elements with attributes

Learn more about all available HTML elements on the MDN HTML element

CSS Summary

CSS(Cascading Style Sheet) for styling your page content.

    Styles are assigned via property-value pairs

    You can also style via "style" attribute

  • To avoid duplications, you typically use global styles instead

    Alternatively, you can work with external style sheet files which you link to

  • When working with CSS concepts like inheritance, "specificility" and "box model" should be understood

Learn more about all available CSS properties and values on the MDN CSS element
